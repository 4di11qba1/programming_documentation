HTML Frames

FRAMESET TAG

HTML also allows the webpages to be split into more than one parts. For this purpose we use (frameset)(frame) (/frame)(/frameset) tags.

FrameSet Attributes

AttributeDescription
ROWSDivides the screen into multiple rows. It can be set equal to a list of values. Depending on the required size of each row. The values can be:
  • A number of pixels
  • Expressed as percentage
COLSDivides the screen into multiple columns. It can be set equal to a list of values. Depending on the required size of a each column. The values can be:
  • A number of pixels
  • Expressed as percentage

FRAME TAG

Once the screen is divided into 2 or more parts each frame can open a specific page of a website. This can be achieved by using (frame) (/frame) tag.

Frame Attributes

AttributeDescription
SRC="url"URL of the document
MARGINHEIGHT=nSpecifies the document to be at left at the top and bottom of the page
MARGINWIDTH=nSpecifies the document to be at left along the sides of the page
NAME="name"Gives the frame a unique name so that it can be targeted by other documents
NORESIZEDisables the frame resizing capability
SCROLLINGControls the appearance of horizontal and vertical scrollbars in a frame

The following code demonstrate the use of (frameset) and (frame) tags.

Coding

(frameset rows="30%,*")
(frameset cols="50%, 50%" )
(frame src="filel.html")
(frame src="fi1e2.htm1")
( / frameset)
(frameset cols="50%,50%")
(frame src="file3.html")
(frame src="file4.html")
(/frame)
( / frameset)
( / frameset)

Output

The screen is distributed into 4 frames.
