HTML Links

HTML allows linking to other HTML documents as well as images. Clicking on a section of text or an image in one webpage will open as entire webpage or an image. This text is called HYPERTEXT, a HYPERLINK, or a HOTSPOT.

The browser distinguishes Hyperlinks from normal text. Every Hyperlink,

  • Appears blue in color
  • The hyperlink text/image is underlined
  • When the pointer is placed on it, it changes into the form of a hand

The blue color, which appears by default, can be over-ridden. To change these link colors, there are some attributes.

Linking Attributes

The attibutes for linking can be used with the (body) tag. These are:

AttributeDescription
LinkChanges the color of the hyperlink
ALinkChanges the color of the hyperlink that is activated
VLinkChanges the color of the hyperlink that is already visited

Links are created in a webpage using (a) (/a) tags. Anything written within it becomes a Hyperlink. Following is the syntax of the hyperlink tag.

(a href="filename.html")

Hyperlinks can of two types:

  • links to an external document
  • links to a specific place within the same document

The following example demonstrate the use of (a) (/a) tag:

Coding

(a href="#")Google(/a)
(a href="#")Youtube(/a)
(a href="#")Facebook(/a)

Output

Google Youtube Facebook
