HTML Tables

A table is a two dimensional matrix consisting of rows and columns. Tables are intended for displaying data in columns on a webpage. All table related tags are included between the (table) (/table) tags. Each row of a table is described between the (tr) (/tr) tags. Each column of a table is described between the (td) (/td) tags.

Table Attributes

The attributes that can be included in a (table) tag are:

AttributeDescription
ALIGNHorizontal alignment is controlled by the ALIGN attribute. It can be set to LEFT,CENTER or RIGHT
VALIGNVertical alignment is controlled by the VALIGN attribute. It can be set to TOP,MIDDLE or BOTTOM
WIDTHSets the width of the table to a specific number of pixels or percentage.
BORDERControls the border to be placed around the table.
CELLPADDINGControls the distance between the data in a cell and the boundaries of the cell.
CELLSPACINGControls the distance between rows and columns.
COLSPANControls the number of columns that a specific cell will occupy.
ROWSPANControls the number of columns that a specific cell will occupy.

The Caption Tag

Captions are the headings that represent the type of information in the table. Captions are the given inside (caption) (/caption) tags.
